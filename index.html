基礎理論

三角関数の基礎

sin・cos・tan

三角関数は円周上の点の座標を表します：

  • sin(θ) = y座標
  • cos(θ) = x座標
  • tan(θ) = sin(θ)/cos(θ)

JavaScriptでの実装方法

角度から座標を計算：

// 角度をラジアンに変換
const angleRadians = (angleDegrees * Math.PI) / 180;

// 三角関数で座標を計算
const x = centerX + radius * Math.cos(angleRadians);
const y = centerY - radius * Math.sin(angleRadians); // Y軸は反転

角度: 0°

線形代数の基礎

座標変換

2D座標変換の基本操作：

  • 平行移動: (x, y) → (x + dx, y + dy)
  • 回転: 角度θで回転
  • 拡大縮小: スケール係数sで拡大

回転行列の実装

回転行列の計算：

// 角度をラジアンに変換
const angleRadians = (angleDegrees * Math.PI) / 180;

// 回転行列の要素
const cos = Math.cos(angleRadians);
const sin = Math.sin(angleRadians);

// 回転後の座標を計算
const newX = x * cos - y * sin;
const newY = x * sin + y * cos;

回転角度: 0°

基本的な描画操作

HTML5 Canvas APIを使用した図形の描画：

  • 円の描画（三角関数活用）
  • 多角形の描画
  • アニメーションの実装

Canvas APIでの描画実装

正多角形の描画：

// 正六角形を描画
ctx.beginPath();
for (let i = 0; i < 6; i++) {
    const angle = (i * Math.PI) / 3;
    const x = centerX + radius * Math.cos(angle);
    const y = centerY + radius * Math.sin(angle);
    if (i === 0) {
        ctx.moveTo(x, y);
    } else {
        ctx.lineTo(x, y);
    }
}
ctx.closePath();
ctx.fill();

応用例

図形描画

マウス操作による図形の制御

マウスの位置に応じて図形が変化します：

  • マウス位置による回転
  • 距離に応じたスケール変化
  • リアルタイムな視覚的フィードバック

インタラクティブ処理の実装

マウス位置の計算と図形制御：

// マウスと中心の距離を計算
const dx = mouseX - centerX;
const dy = mouseY - centerY;
const distance = Math.sqrt(dx * dx + dy * dy);

// 角度を計算
const angle = Math.atan2(dy, dx);

// 距離に応じたスケール
const scale = Math.min(2, Math.max(0.5, distance / 100));

// 図形の回転とスケール
ctx.save();
ctx.translate(centerX, centerY);
ctx.rotate(angle);
ctx.scale(scale, scale);

波動アニメーションの実装

sin関数を使った波動の生成：

// 波動アニメーション
for (let x = 0; x < canvas.width; x++) {
    const normalizedX = (x / canvas.width) * Math.PI * 4;
    const amplitude = 30 + wave * 10;
    const frequency = 1 + wave * 0.5;
    const phase = time + wave * Math.PI / 3;
    
    const y = centerY + amplitude * Math.sin(frequency * normalizedX + phase);
    
    if (x === 0) {
        ctx.moveTo(x, y);
    } else {
        ctx.lineTo(x, y);
    }
}

波の数: 3
